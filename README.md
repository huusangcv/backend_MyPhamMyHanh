# API

API for webiste https://myphammyhanh.regis.id.vn/

## Run Locally

Clone the project

```bash
  git clone https://github.com/huusangcv/backend_MyPhamMyHanh
```

Go to the project directory

```bash
  cd backend_MyPhamMyHanh
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
